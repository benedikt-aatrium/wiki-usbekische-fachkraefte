# 06 Checklisten

> **Ziel:** Praktische Checklisten für Arbeitgeber, Kandidat und AATRIUM/Recruiter.

**Stand:** 2026-07-10

---

## 6.1 Checkliste für AATRIUM/Recruiter (pro Kandidat)

### Vor Vertrag

- [ ] Pass kopieren und Gültigkeit prüfen (mindestens 3 Monate über Aufenthalt hinaus)
- [ ] Lebenslauf lückenlos und tabellarisch auf Deutsch anfertigen
- [ ] Ausbildungsnachweise sammeln: Diplom, Ilova, Transkript, Jahreszeugnisse
- [ ] Berufserfahrung nachweisen: Arbeitsbuch / Arbeitsbescheinigung / Zeugnisse
- [ ] Sprachnachweise prüfen (Deutsch B1, ggf. Englisch/Russisch)
- [ ] Referenzberuf festlegen (zusammen mit Arbeitgeber)
- [ ] Zuständige Anerkennungsstelle klären (HWK/IHK FOSA/ZAB)
- [ ] Zuständige HWK für den Standort des Arbeitgebers ermitteln
- [ ] Ausbildung/Qualifikation grob einschätzen: Passt der Kandidat zum Referenzberuf?
- [ ] Stellenbeschreibung und Arbeitsbedingungen prüfen
- [ ] Prüfen, ob Anerkennung vor Visum nötig ist oder später möglich

### Nach Vertrag

- [ ] Arbeitsvertrag prüfen (Pflichtangaben, Mindestlohn, Arbeitszeit, Kündigungsfristen)
- [ ] Zustimmung BA/ZAV prüfen und beantragen
- [ ] Kandidat informieren, welche Dokumente er benötigt
- [ ] Übersetzungen organisieren (je nach Anerkennungsstelle)
- [ ] Apostille prüfen, falls Visa-Stelle verlangt
- [ ] Krankenversicherung für Einreise klären
- [ ] Unterkunft in Deutschland organisieren oder Nachweis beschaffen

### Visum

- [ ] Visum-Termin bei TLScontact/Botschaft Taschkent buchen
- [ ] Visumsantragsformular prüfen und begleiten
- [ ] Dokumentenmappe zusammenstellen (Pass, Fotos, Vertrag, Zustimmung, Ausbildungsnachweise, Krankenversicherung, Lebenslauf, Unterkunftsnachweis)
- [ ] Visumgebühr bezahlen
- [ ] Bearbeitungsstatus verfolgen

### Nach Einreise

- [ ] Meldung beim Einwohnermeldeamt innerhalb 14 Tage
- [ ] Ausländerbehörde / Aufenthaltstitel beantragen
- [ ] Sozialversicherungsnummer beantragen
- [ ] Krankenversicherung abschließen
- [ ] Bankkonto eröffnen
- [ ] Anerkennung parallel/fortführen, falls noch nicht abgeschlossen
- [ ] Arbeitgeber über Einreise und Verfügbarkeit informieren

---

## 6.2 Checkliste für den Arbeitgeber

- [ ] Stellenbeschreibung mit Tätigkeitsschwerpunkt, Standort, Arbeitszeit, Vergütung vorliegen
- [ ] Arbeitsvertrag mit allen Pflichtangaben nach deutschem Recht
- [ ] Tarifliche/ortsübliche Vergütung sicherstellen
- [ ] Unterkunft für den Kandidaten organisieren oder Nachweis liefern
- [ ] Zustimmung BA/ZAV frühzeitig beantragen (oder bevollmächtigen)
- [ ] Ausbildungsnachweise und Lebenslauf des Kandidaten prüfen
- [ ] Krankenversicherung für die ersten Wochen klären
- [ ] Betriebsarzt/Arbeitsmedizinische Vorsorge ggf. planen
- [ ] Sozialversicherungsmeldung vorbereiten
- [ ] Ansprechpartner für Behörden in der Firma benennen

---

## 6.3 Checkliste für den Kandidaten

- [ ] Reisepass gültig? (Mind. 3 Monate über Aufenthalt hinaus + 2 leere Seiten)
- [ ] Passfotos aktuell (biometrisch, nicht älter als 6 Monate)
- [ ] Diplom, Ilova, Transkript, Jahreszeugnisse zusammengestellt
- [ ] Arbeitsbuch / Arbeitsbescheinigung / Zeugnisse zusammengestellt
- [ ] Sprachnachweise (Deutsch B1) vorhanden
- [ ] Lebenslauf lückenlos auf Deutsch
- [ ] Krankenversicherung für Reise/erste Wochen geklärt
- [ ] Wohnadresse in Deutschland bekannt
- [ ] Visum-Termin gebucht
- [ ] Unterlagen für Botschaft/TLScontact vollständig
- [ ] Meldung und Aufenthaltstitel nach Einreise organisiert

---

## 6.4 Dokumenten-Checkliste für die Anerkennung

### Für HWK (Handwerk)

- [ ] Antragsformular
- [ ] Passkopie
- [ ] Tabellarischer Lebenslauf
- [ ] Farbkopie Diplom + deutsche Übersetzung (deutscher Übersetzer bevorzugt)
- [ ] Farbkopie Ilova/Anlage + deutsche Übersetzung
- [ ] Farbkopie Transkript/Jahreszeugnisse + deutsche Übersetzung
- [ ] Arbeitszeugnisse/Arbeitsbuch + deutsche Übersetzung
- [ ] Ggf. weitere Befähigungsnachweise

### Für IHK FOSA

- [ ] Antragsformular IHK FOSA
- [ ] Passkopie
- [ ] Tabellarischer Lebenslauf
- [ ] Farbkopien der Originaldokumente (vollständig, gut lesbar)
- [ ] Diplom + Übersetzung (deutscher oder usbekischer Übersetzer möglich)
- [ ] Ilova/Transkript mit Stunden und Noten
- [ ] Arbeitszeugnisse/Berufserfahrung
- [ ] Nachweis der Erwerbsabsicht

### Für ZAB (Hochschulabschluss)

- [ ] Abschlusszeugnis in Originalsprache
- [ ] Fächerübersicht/Transkript
- [ ] Übersetzung nach Vorgabe der ZAB
- [ ] Nachweis der Hochschulzugehörigkeit

---

## 6.5 Dokumenten-Checkliste für das Visum (Vorlage)

- [ ] Visumantragsformular (ausgefüllt und unterschrieben)
- [ ] Zwei aktuelle Passfotos
- [ ] Gültiger Reisepass + Kopie
- [ ] Kopie aller bisherigen Visa/Stempel
- [ ] Arbeitsvertrag / Arbeitsplatzangebot
- [ ] Zustimmung BA/ZAV (sofern erforderlich)
- [ ] Arbeitsplatzbeschreibung
- [ ] Diplom + Ilova/Transkript (Original oder beglaubigte Kopie)
- [ ] Deutsche Übersetzungen (ggf. beglaubigt)
- [ ] Arbeitszeugnisse/Berufserfahrung
- [ ] Lückenloser tabellarischer Lebenslauf (deutsch)
- [ ] Krankenversicherungsnachweis
- [ ] Unterkunftsnachweis in Deutschland
- [ ] Visumgebühr bezahlt
- [ ] Ggf. Apostille/Beglaubigung für Dokumente

> **Wichtig:** Diese Visum-Checkliste ist eine Vorlage. Die aktuelle Checkliste der Botschaft Taschkent / TLScontact ist maßgebend.
