# Kostenübersicht für usbekische Fachkräfte

## Kurz erklärt

Der Weg einer usbekischen Fachkraft nach Deutschland verursacht verschiedene Kosten. Eine frühzeitige Planung und klare Absprachen mit dem Arbeitgeber sind wichtig.

---

## Kosten im Herkunftsland (Usbekistan)

| Position | Geschätzte Kosten | Hinweis |
|----------|-------------------|---------|
| Deutschkurs A1–B1 | 300–1.500 € | Je nach Intensität und Anbieter |
| Sprachprüfung (Goethe/ÖSD/TELC) | 100–250 € | Pro Prüfung |
| Übersetzungen Zeugnisse | 50–200 € | Pro Dokument |
| Beglaubigte Übersetzungen | 80–250 € | Pro Dokument |
| Apostille / Legalisation | 20–100 € | Pro Dokument |
| Arbeitsvermittlung / Agentur | 0–5.000 € | Stark variabel |

---

## Kosten in Deutschland (vor Arbeitsbeginn)

| Position | Geschätzte Kosten | Hinweis |
|----------|-------------------|---------|
| Visumgebühr | 75–100 € | Nationalvisum D |
| Anerkennung IHK FOSA | ca. 100–300 € | Je nach Beruf; Details Artikel 18 |
| Anerkennung HWK | ca. 100–500 € | Je nach Kammer; Details Artikel 18 |
| Führerschein-Umschreibung | 500–1.500 € | Inkl. Fahrstunden, Prüfung |
| Krankenversicherung | 0 € | Über Arbeitgeber ab erstem Arbeitstag |
| Bankkonto | 0–10 €/Monat | Je nach Bank |
| Erstausstattung Wohnung | 500–2.000 € | Bett, Küche, Kleidung |
| Reisekosten Usbekistan → Deutschland | 300–800 € | Flug |

## Laufende Kosten

| Position | Geschätzte Kosten | Hinweis |
|----------|-------------------|---------|
| Miete | 300–800 €/Monat | Je nach Region |
| Krankenkasse | 14,6 % vom Brutto gesamt (je 7,3 % Arbeitnehmer + Arbeitgeber) + Zusatzbeitrag | Arbeitnehmer zahlt nur die Hälfte |
| Rentenversicherung | 18,6 % vom Brutto gesamt (je 9,3 % AN + AG) | Arbeitnehmer zahlt nur die Hälfte |
| Arbeitslosenversicherung | 2,4 % vom Brutto gesamt (je 1,2 % AN + AG) | Arbeitnehmer zahlt nur die Hälfte |
| Pflegeversicherung | 3,4 % vom Brutto gesamt (je 1,7 % AN + AG) | Kinderlose zahlen Zuschlag |
| Steuern | variabel | Lohnsteuer abhängig von Bruttoeinkommen |
| Integrationskurs | 1,95 €/Stunde | ca. 1.170 € Gesamtkosten (600 Stunden); kostenlos für Arbeitsuchende; Details siehe Artikel 25 |

## Wer trägt die Kosten?

| Kostenblock | Typische Regelung |
|-------------|-------------------|
| Sprachkurs im Herkunftsland | Arbeitgeber oder Fachkraft |
| Visum- und Reisekosten | Arbeitgeber oder Fachkraft |
| Anerkennung | Arbeitgeber, Fachkraft oder geteilt |
| Führerschein | Arbeitgeber oder Fachkraft |
| Wohnung | Arbeitgeber stellt Unterkunft oder Fachkraft zahlt Miete |
| Sozialversicherung | Arbeitgeber und Arbeitnehmer je zur Hälfte |

---

## Praktische Tipps für AATRIUM

- Legen Sie im Arbeitsvertrag fest, welche Kosten der Arbeitgeber übernimmt.
- Planen Sie **Vorschlussgeld** für die ersten 1–2 Monate ein, bis das Gehalt fließt.
- Vereinbaren Sie eine **Darlehens- oder Rückzahlungsregelung** bei Vorschusszahlungen.
- Dokumentieren Sie alle Kosten und Zahlungen schriftlich.
